# NCDBRecode
This package recodes the standard NCDB excel or stata .sav file for ease of analysis
Function takes a dataframe of default NCDB nomenclature and returns same.
Definitions built on PUF 2016 data dictionary (cases spanning 2004-2016) [http://ncdbpuf.facs.org/]
Revisions to the PUF user file can be found here [http://ncdbpuf.facs.org/?q=node/376]
